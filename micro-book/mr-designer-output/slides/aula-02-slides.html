Revisão Relâmpago: O Lagrangeano

Problema canônico de otimização com restrição de igualdade:

\[\max_{x_1, x_2} f(x_1, x_2) \quad \text{s.a.} \quad g(x_1, x_2) = c\]

Lagrangeano:

\[\mathcal{L} = f(x_1, x_2) - \lambda\big[g(x_1, x_2) - c\big]\]

Condições de Primeira Ordem (CPOs):

\[\frac{\partial \mathcal{L}}{\partial x_i} = \frac{\partial f}{\partial x_i} - \lambda \frac{\partial g}{\partial x_i} = 0, \quad i = 1, 2\]

\[\frac{\partial \mathcal{L}}{\partial \lambda} = -(g(x_1, x_2) - c) = 0\]

Intuição Econômica

O multiplicador \(\lambda\) mede o “preço sombra” da restrição — quanto o valor ótimo melhoraria se relaxássemos a restrição em uma unidade.

Teorema do Envelope

Dado o valor ótimo como função do parâmetro \(\alpha\):

\[V(\alpha) = \max_{x} f(x, \alpha) \quad \text{s.a.} \quad g(x, \alpha) = 0\]

Teorema do Envelope:

\[\frac{dV}{d\alpha} = \frac{\partial \mathcal{L}}{\partial \alpha}\bigg|_{x = x^*(\alpha)}\]

Por que é poderoso?

Não precisamos resolver \(x^*(\alpha)\) explicitamente para saber como \(V\) muda com \(\alpha\).

Exemplo clássico:

Para \(V(m) = \max U\) s.a. orçamento \(= m\):

\[\frac{dV}{dm} = \lambda^*\]

O multiplicador é a utilidade marginal da renda.

Envelope em Ação

Considere \(U = xy\) sujeito a \(x + y = c\). A solução é \(x^* = y^* = c/2\), logo \(V^*(c) = c^2/4\) e \(\lambda^* = c/2\).

Intuição Econômica

A reta tangente à curva \(V^*(c)\) tem inclinação \(\lambda^*\). O Teorema do Envelope diz que a derivada do valor ótimo em relação ao parâmetro é exatamente o multiplicador de Lagrange avaliado no ótimo.

Condições de Kuhn-Tucker (KKT)

Quando a restrição é uma desigualdade: \(g(x_1, x_2) \leq c\)

\[\mathcal{L} = f(x_1, x_2) - \lambda\big[g(x_1, x_2) - c\big]\]

Condições KKT:

  1. \(\dfrac{\partial \mathcal{L}}{\partial x_i} = 0\) para todo \(i\)
  2. \(\lambda \geq 0\) (multiplicador não-negativo)
  3. \(g(\mathbf{x}) \leq c\) (viabilidade primal)
  4. \(\lambda\big[g(\mathbf{x}) - c\big] = 0\) (folga complementar)

Intuição Econômica

As condições KKT generalizam o Lagrangeano clássico para restrições de desigualdade. A novidade central é a restrição de sinal sobre \(\lambda\) e a condição de folga complementar.

KKT: Folga Complementar

A condição \(\lambda\big[g(\mathbf{x}) - c\big] = 0\) implica dois cenários mutuamente exclusivos:

Caso 1: Restrição ativa (\(g = c\))

  • A restrição “morde”
  • \(\lambda^* > 0\) (tem custo sombra)
  • Relaxar a restrição melhoraria o objetivo
  • Solução está na fronteira

Exemplo: consumidor gasta toda a renda

Caso 2: Restrição inativa (\(g < c\))

  • A restrição “sobra”
  • \(\lambda^* = 0\) (sem custo sombra)
  • Relaxar a restrição não muda nada
  • Solução é o ótimo irrestrito

Exemplo: restrição de capacidade com folga

Regra prática: Resolva assumindo \(\lambda > 0\) (restrição ativa). Se obtiver \(\lambda < 0\), descarte e resolva o caso irrestrito com \(\lambda = 0\).

KKT Interativo

Observe como a solução muda conforme a restrição se torna ativa ou inativa:

Intuição Econômica

Quando \(c\) é grande o suficiente, o ótimo irrestrito já satisfaz \(g \leq c\) e o multiplicador \(\lambda^*\) cai a zero. A restrição só tem “preço” quando efetivamente limita a escolha.

Exemplo Resolvido: Utilidade Cobb-Douglas

\[\max_{x,y} \; U = x^{0{,}5}\, y^{0{,}5} \quad \text{s.a.} \quad 2x + 3y = 120\]

Passo 1 — Lagrangeano:

\[\mathcal{L} = x^{0{,}5}\,y^{0{,}5} - \lambda(2x + 3y - 120)\]

Passo 2 — CPOs:

\[\frac{0{,}5\, y^{0{,}5}}{x^{0{,}5}} = 2\lambda \qquad \frac{0{,}5\, x^{0{,}5}}{y^{0{,}5}} = 3\lambda\]

Passo 3 — Dividindo CPO₁ por CPO₂:

\[\frac{y}{x} = \frac{2}{3} \implies y = \frac{2x}{3}\]

Passo 4 — Substituindo na restrição:

\[2x + 3 \cdot \frac{2x}{3} = 120 \implies 4x = 120 \implies x^* = 30, \quad y^* = 20\]

Exemplo Resolvido: Solução de Canto

\[\max_{x,y} \; U = x + 2y \quad \text{s.a.} \quad x^2 + y^2 \leq 25, \quad x \geq 0, \quad y \geq 0\]

Lagrangeano com KKT:

\[\mathcal{L} = x + 2y - \lambda(x^2 + y^2 - 25)\]

CPOs: \(1 = 2\lambda x\) e \(2 = 2\lambda y\)

Da razão: \(\dfrac{1}{2} = \dfrac{x}{y} \implies y = 2x\)

Substituindo: \(x^2 + 4x^2 = 25 \implies x^* = \sqrt{5}, \; y^* = 2\sqrt{5}\)

Verificação KKT: \(\lambda^* = \frac{1}{2\sqrt{5}} > 0\) — restrição ativa. O ótimo irrestrito (\(U\) crescente) seria ilimitado, então a restrição de capacidade necessariamente “morde”.

Gallery Walk — Instruções

Formato: Gallery Walk (20 min)

Instruções:

  1. Formem grupos de 3–4 pessoas
  2. Cada estação tem um problema de otimização diferente
  3. Resolvam na folha da estação (8 min), depois rodem
  4. Ao final, cada grupo apresenta a solução da estação onde terminou

Estações:

Estação Problema Ferramenta
A Consumidor Cobb-Douglas com 3 bens Lagrangeano
B Firma com restrição de capacidade KKT
C Governo com orçamento limitado KKT + Envelope

Dica: Para cada problema, identifiquem primeiro se a restrição é de igualdade ou desigualdade!

Problema 1: Consumidor Cobb-Douglas

Estação A — Lagrangeano clássico

\[\max_{x,y,z} \; U = x^{1/3}\,y^{1/3}\,z^{1/3} \quad \text{s.a.} \quad 2x + 4y + 5z = 200\]

Roteiro:

  1. Escreva o Lagrangeano \(\mathcal{L}\)
  2. Obtenha as 4 CPOs (3 variáveis + restrição)
  3. Use a razão das CPOs para encontrar as relações entre \(x\), \(y\) e \(z\)
  4. Substitua na restrição e resolva
  5. Calcule \(\lambda^*\) e interprete

Intuição Econômica

Com Cobb-Douglas simétrica (expoentes iguais), o consumidor gasta frações iguais da renda em cada bem. Verifique: \(p_x x^* = p_y y^* = p_z z^*\)?

Problema 2: Firma com Restrição de Capacidade

Estação B — Kuhn-Tucker

\[\max_{L} \; \pi = 10\sqrt{L} - 2L \quad \text{s.a.} \quad L \leq 16\]

Roteiro:

  1. Resolva o problema irrestrito: \(\pi'(L) = 0\)
  2. Verifique se o ótimo irrestrito satisfaz \(L \leq 16\)
  3. Se sim, \(\lambda^* = 0\) e acabou. Se não, resolva com restrição ativa (\(L = 16\))
  4. Qual é o valor de \(\lambda^*\)? Interprete

Bônus: Use o Teorema do Envelope para determinar \(\frac{d\pi^*}{dc}\) onde \(c\) é o limite de capacidade.

Problema 3: Governo com Orçamento

Estação C — KKT + Envelope

\[\max_{x,y} \; W = \ln(x) + \ln(y) \quad \text{s.a.} \quad 3x + 5y \leq B, \quad B = 80\]

Roteiro:

  1. Monte o Lagrangeano e obtenha as CPOs
  2. Resolva para \(x^*(B)\) e \(y^*(B)\)
  3. Calcule \(\lambda^*\) e interprete: quanto vale um real a mais no orçamento?
  4. Use o Teorema do Envelope para verificar: \(\frac{dW^*}{dB} = \lambda^*\)

Intuição Econômica

Com utilidade logarítmica, o multiplicador \(\lambda^*\) é decrescente em \(B\) — o “valor” de uma unidade extra de orçamento é maior quando o orçamento é apertado.

Demandas CES via Lagrangeano

A função de utilidade CES: \(U = \big(x^{\rho} + y^{\rho}\big)^{1/\rho}\), com elasticidade de substituição \(\sigma = \frac{1}{1-\rho}\).

Intuição Econômica

A elasticidade de substituição \(\sigma\) governa quão “curvadas” são as curvas de demanda. Quanto maior \(\sigma\), mais o consumidor substitui entre bens quando o preço relativo muda.

Funções Homogêneas — Propriedades Úteis

\(f\) é homogênea de grau \(k\) se \(f(t\mathbf{x}) = t^k f(\mathbf{x})\), \(\forall t > 0\).

Teorema de Euler:

\[\sum_{i=1}^n x_i \frac{\partial f}{\partial x_i} = k \cdot f(\mathbf{x})\]

Aplicações em Microeconomia:

  • Produção CRS (\(k=1\)): pagando cada fator pelo produto marginal, esgota-se a receita total
  • Demandas Marshallianas: homogêneas de grau 0 em \((p, m)\) — sem ilusão monetária
  • Funções custo: homogêneas de grau 1 em preços dos fatores

Consequências úteis:

  • Derivadas de \(f\) hom. grau \(k\) são hom. grau \(k-1\)
  • Se \(k=1\): \(\sum x_i f_i = f\) (Exaustão)
  • Se \(k=0\): \(\sum x_i f_i = 0\) (derivadas somam zero ponderadas)

Intuição Econômica

Se a função de produção tem retornos constantes de escala (\(k=1\)), pagar cada fator pelo seu produto marginal esgota exatamente a receita total — é o Teorema da Exaustão.
