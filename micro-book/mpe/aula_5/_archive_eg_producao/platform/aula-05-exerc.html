Gabarito 5-passos — Exercício 1

1. Ponto-chave. Robinson Crusoe canônico: 1 consumidor + 1 firma. Mercado descentralizado replica solução do planejador. EC tem \(\text{TMS} = \text{TMT} = w/p\). Verificação numérica de toda a maquinaria: planejador, max-lucro, UMP, fechamento de mercados.

2. Derivação. Planejador: \(\max_L L^{1/4}(16-L)^{1/2}\). CPO: derivar e igualar a zero entrega \((16-L) - 2L = 0 \implies L^* = 16/3\). ✓ (a). Da firma: \(f'(L^*) = w/p \Rightarrow w^* = 1/(2\sqrt{16/3}) = \sqrt{3}/8\). ✓ (b). Lucro: \(\pi^* = p\, q^* - w^*\, L^* = 4/\sqrt{3} - (\sqrt{3}/8)(16/3) = 4/\sqrt{3} - 2/\sqrt{3} = 2\sqrt{3}/3\). ✓ (c). Renda: \(m^* = w^*\, \bar L + \pi^* = (\sqrt{3}/8)\cdot 16 + 2\sqrt{3}/3 = 2\sqrt{3} + 2\sqrt{3}/3 = 8\sqrt{3}/3\). ✓ (d).

4. Extensão. Verificação \(\text{TMS} = \text{TMT} = w/p\): \(\text{TMS} = q^*/\ell^* = (4/\sqrt{3})/(32/3) = \sqrt{3}/8\). \(\text{TMT} = f'(L^*) = \sqrt{3}/8\). \(w^*/p^* = \sqrt{3}/8\). Os três coincidem. Walras com produção: mercado de trabalho fecha (\(L^* = 16/3\)) e mercado de bem fecha (\(q^* = \sqrt{16/3}\)).

Gabarito 5-passos — Exercício 3

1. Ponto-chave. Robinson Crusoe com peso assimétrico no consumo. Lição-chave: a composição \(q = f(L) = L^{1/2}\) "mata" o peso \(2/3\) da utilidade na fórmula reduzida. Função objetivo final \(L^{1/3}(9-L)^{1/3}\) é simétrica, ótimo em \(L^* = 9/2\). Quem tenta intuir "trabalha mais porque ama consumir" sem fazer a álgebra erra.

2. Derivação. Substituir \(q = L^{1/2}\), \(\ell = 9 - L\): \(u(L) = L^{1/3}(9-L)^{1/3}\). CPO entrega \((9-L) - L = 0 \implies L^* = 4{,}5\). ✓ (a). \(\ell^* = 4{,}5\). ✓ (b). \(q^* = \sqrt{9/2} = 3/\sqrt{2}\). ✓ (c). \(w^* = 1/(2\sqrt{9/2}) = \sqrt{2}/6\). ✓ (d). \(\text{TMS}_{\ell, q} = (\alpha_\ell/\alpha_q)(q/\ell) = (1/2)(q/\ell) = (1/2)(3/\sqrt{2})/(9/2) = 1/(3\sqrt{2}) = \sqrt{2}/6\). \(\text{TMT} = f'(L^*) = \sqrt{2}/6\). \(w^*/p^* = \sqrt{2}/6\). ✓ (e).

3. Armadilha. (a)(iii) \(L^* = 6\) é o "intuitivo errado" — quem pensa "peso 2/3 ⇒ trabalha 2/3 do tempo" cai aqui. (a)(iv) é o oposto. (c)(ii) "\(q^* = 3\)" é alguém que substituiu \(L = 9\). (d)(i) confunde Robinson com Aula 4. (e)(ii) ignora que TMS = TMT em qualquer EC interior por construção.

4. Extensão. Fórmula geral: com \(u = q^\alpha\ell^{1-\alpha}\) e \(f(L) = L^\beta\), \(L^* = \alpha\beta\bar L/(\alpha\beta + 1 - \alpha)\). Em \(\alpha = 2/3, \beta = 1/2\): \(L^* = (1/3)\bar L/(1/3 + 1/3) = \bar L/2 = 4{,}5\). ✓ Para \(\beta = 1\) (linear) e mesmo \(\alpha = 2/3\): \(L^* = (2/3)\bar L = 6\) — aí a intuição funciona. A interação preferência × tecnologia é o ponto.

5. Peer-review. Calibre 🟡. Distrator (a)(iii) \(L^* = 6\) é deliberado — quem usa intuição em vez de álgebra cai. Calibração honesta de "intuição" vs. "cálculo" — exatamente o que Aula 5 quer corrigir.

1. Ponto-chave. A prova do 1º TBE com produção tem 4 passos: contradição → passo 1 (UMP para consumidor estritamente melhor) → passo 2 (LNS para consumidores fracamente melhores) → passo 3 (max-lucro para firmas) → soma e contradiz viabilidade. Gabarito 5-passos — Exercício 5

1. Ponto-chave. Eficiência produtiva = TMT igual a razão de preços relativos. Sem distorção, mercado descentralizado replica isso via maximização de lucro com mobilidade de fator. Distorção tributária faz a TMT-produção se afastar da TMS-consumo, gerando wedge e ineficiência.

2. Derivação. Da firma \(\ell\): \(\max p_\ell f_\ell(L_\ell) - w L_\ell\), CPO \(p_\ell f_\ell' = w \implies f_\ell' = w/p_\ell\). ✓ (a). \(\text{TMT}_{\ell k} = -dq_k/dq_\ell = (df_k/dL_k)(dL_k/dq_\ell) \cdot (-1)\): regra da cadeia com \(L_k = \bar L - L_\ell\), \(L_\ell = f_\ell^{-1}(q_\ell)\) entrega \(\text{TMT}_{\ell k} = \text{PMa}^k/\text{PMa}^\ell\). ✓ (b). Combinando: \(\text{TMT}_{\ell k} = (w/p_k)/(w/p_\ell) = p_\ell/p_k\). ✓ (c). Com tributo: firma 1 vê \(p_1(1-\tau)\), CPO \(f_1' = w/[p_1(1-\tau)]\). \(\text{TMT}_{12} = f_2'/f_1' = (w/p_2)/(w/(p_1(1-\tau))) = p_1(1-\tau)/p_2\). ✓ (d). Mercado de bens enxerga \(p_1/p_2\) (preço de consumidor); wedge \(\text{TMS}_{12} \neq \text{TMT}_{12}\) ⇒ ineficiência. ✓ (e).

3. Armadilha. (a)(i) ignora que CPO da firma é o que **gera** a igualdade. (b)(i) confunde TMT com razão de quantidades — só vale em casos particulares (PPF circular). (c)(i) inverte direção. (d)(i) ignora pass-through do imposto.

4. Extensão. Conexão com Aula 7 (externalidades): se há externalidade tecnológica positiva em setor 1 (aprendizado, network effects), preço de mercado **subestima** valor social ⇒ TMT-mercado \(\neq\) TMT-social. Argumento simétrico ao da tributação. **Conexão com Box ZFM da aula:** subsídio fiscal regional cria distorção análoga — TMT em Manaus \(\neq\) TMT em SP, e Pareto-agregada é violada (a menos que externalidades positivas justifiquem second-best).

Gabarito 5-passos — Exercício 6

1. Ponto-chave. Retornos crescentes em \(f(L) = L^2\) ⇒ conjunto de produção \(Y\) não-convexo ⇒ separação por hiperplano-preço falha ⇒ 2º TBE com produção quebra. 2. Derivação. \(f(L) = L^2\) tem \(f''(L) = 2 > 0\) ⇒ convexa. ✓ (a). \(Y\) é a sub-gráfica; problema é a fronteira: \((0, 0)\) e \((-2, 4)\) estão em \(Y\); combinação convexa \((-1, 2)\) tem \(L = 1\) com \(f(1) = 1 < 2\), fora de \(Y\). ✓ (b).
