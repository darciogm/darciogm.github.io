    DiD Estimation and Results¶

    TWFE event study (baseline)¶

    We first estimate a standard two-way fixed effects event study:

    \[y_{it} = \sum_{k \neq -1} \beta_k \cdot \mathbf{1}[\text{event\_time}_{it} = k] + \mu_i + \lambda_t + \varepsilon_{it}\]

    with municipality (\(\mu_i\)) and year (\(\lambda_t\)) fixed effects, event-time dummies for \(k \in \{-3, -2, 0, +1, +2\}\) (\(k = -1\) omitted), and standard errors clustered by municipality.

    Implementation

    run_event_study.py using linearmodels.PanelOLS

    Callaway-Sant'Anna¶

    We estimate three specifications using the Callaway and Sant'Anna (2021) doubly robust estimator with base_period = "universal":

    Spec Description Sample
    A Basic CS, never-treated comparison Full (561 munis)
    C Late adopters only (cohorts 2023–2024) + never-treated 350 munis
    D Not-yet-treated comparison group Full (561 munis)
