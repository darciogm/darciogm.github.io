\(w(p) = \dfrac{p^{\gamma}}{(p^{\gamma} + (1-p)^{\gamma})^{1/\gamma}}\)
Tversky–Kahneman (1992): \(\gamma^+ = 0{,}61\), \(\gamma^- = 0{,}69\)
\(w^+(p)\) — ganhos
\(w^-(p)\) — perdas
\(w(p) = p\) — ponderação linear (UE)
