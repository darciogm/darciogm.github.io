Figura 3.3 — Comparação de Tipos de Preferências
Cobb-Douglas
\(u = x_1^{0.5} x_2^{0.5}\)
Hiperboles convexas, TMS decrescente, \(\sigma = 1\)
Substitutos Perfeitos
\(u = x_1 + x_2\)
Retas paralelas, TMS constante, \(\sigma = \infty\)
Complementos Perfeitos
\(u = \min\{x_1, x_2\}\)
Ângulo reto (L), TMS indefinida no vértice, \(\sigma = 0\)
Quase-linear
\(u = \sqrt{x_1} + x_2\)
Translações verticais, TMS depende so de \(x_1\)
