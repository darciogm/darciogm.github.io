Q1. 🟡 Um consumidor tem \(u(x_1, x_2) = x_1^{0{,}4}\,x_2^{0{,}6}\) com preços \(p_1 = 2, p_2 = 3\) e renda \(m = 30\). Qual é o nível de utilidade indireta \(v(p, m)\)?

Gabarito: (b).

1. Ponto-chave. Em CD, calcular \(v\) em dois passos: (i) achar a Marshalliana pela fórmula fechada; (ii) avaliar \(u\) no ponto ótimo.

2. Derivação. \(x_1^M = 0{,}4\cdot 30/2 = 6\); \(x_2^M = 0{,}6\cdot 30/3 = 6\). Walras: \(2(6) + 3(6) = 30\). ✓ Utilidade: \(v = 6^{0{,}4}\cdot 6^{0{,}6} = 6^{0{,}4+0{,}6} = 6^1 = 6\). ✓

3. Armadilha. (a) confunde utilidade com renda — erro de quem acha que "ganhar mais = utilidade igual a renda". A utilidade é ordinal, não a "moeda da renda". (c) é \(m/2 = 15\) — outra confusão aritmética. (d) inventa.

4. Extensão. A fórmula fechada da indireta CD: \(v(p, m) = m\cdot \alpha^\alpha (1-\alpha)^{1-\alpha}/(p_1^\alpha p_2^{1-\alpha})\). Verificar com esses números: constante \(K = 0{,}4^{0{,}4}\cdot 0{,}6^{0{,}6} \approx 0{,}696\cdot 0{,}736 \approx 0{,}512\). Denominador \(p_1^{0{,}4}\cdot p_2^{0{,}6} = 2^{0{,}4}\cdot 3^{0{,}6} \approx 1{,}320\cdot 1{,}933 \approx 2{,}551\). \(v = 30\cdot 0{,}512/2{,}551 \approx 6{,}02\). ✓ Ambos os caminhos batem.

5. Peer-review. A questão admite dois caminhos: (i) direto via \(x^M\) e substituição; (ii) fórmula fechada. O segundo é mais rápido em exercício de papel; o primeiro é mais transparente. Alunos cadentes ao cálculo "de cor" perdem tempo — ensinar ao qual \(\alpha\) liga \(p_i\) é um passe-de-mágica que economiza minutos por prova.

Q2. 🟡 Para \(u(x_1, x_2) = x_1^{1/4}\,x_2^{3/4}\) com preços \(p_1 = 2, p_2 = 4\) e utilidade-alvo \(\bar u = 8\), a função gasto pode ser escrita como \(e(p, \bar u) = K\,\bar u\, p_1^{1/4}\, p_2^{3/4}\) com \(K = 1/((1/4)^{1/4}(3/4)^{3/4}) \approx 1{,}755\). Qual é a Hicksiana \(h_2(p, \bar u)\)?

Gabarito: (c).

1. Ponto-chave. Shephard em Cobb-Douglas: \(h_i = \alpha_i\,e/p_i\). Aplicar a fórmula direta em vez de re-derivar.

2. Derivação. \(e(p, 8) = 1{,}755\cdot 8\cdot 2^{1/4}\cdot 4^{3/4} = 14{,}04\cdot 1{,}189\cdot 2{,}828 \approx 47{,}20\). \(h_2 = (3/4)\cdot e/p_2 = 0{,}75\cdot 47{,}20/4 \approx 8{,}85\). Verificação: \(h_1 = (1/4)\cdot e/p_1 = 0{,}25\cdot 47{,}20/2 \approx 5{,}90\). Confere \(u(h_1, h_2) = 5{,}90^{1/4}\cdot 8{,}85^{3/4} \approx 1{,}558\cdot 5{,}13 \approx 8{,}00 = \bar u\). ✓ Walras: \(2(5{,}90) + 4(8{,}85) = 11{,}80 + 35{,}40 = 47{,}20\). ✓

3. Armadilha. (a) é chute "baixo" baseado no preço. (b) é distrator estrutural (\(\bar u - 2\), sem sentido). (d) é \(p_1\cdot h_1\) — gasto em bem 1, não quantidade de bem 2; confunde dimensões.

4. Extensão. Repare que \(h_2 > h_1\) porque a parcela em bem 2 é maior (\(\alpha_2 = 3/4\)) e o preço relativo \(p_2/p_1 = 2\) não é alto o suficiente para inverter. Regra geral em CD: a razão de quantidades \(h_2/h_1 = (\alpha_2/\alpha_1)(p_1/p_2) = (3/1)(1/2) = 3/2 = 8{,}85/5{,}90\). ✓

5. Peer-review. Questão mecânica de aplicação. Se o aluno chegou em \(h_2 \approx 8{,}85\) pela fórmula Shephard direta em menos de 90s, domina. Se fez tangência + CPO do EMP, chegou mas demorou — é aceitável, mas Shephard atalha. Sanidade dimensional: \(h_i\) tem unidade "quantidade"; qualquer resposta em "moeda" (como 11,80) falha no teste.

Q3. 🟡 Um consumidor tem \(u(x_1, x_2) = x_1^{1/2}\,x_2^{1/2}\), \(p_1 = 2, p_2 = 8\) e \(m = 16\). Calcule \(v(p, m)\); depois, com \(\bar u = v\), calcule \(e(p, \bar u)\). Confirme \(e = m\). Marque a afirmação correta:

Gabarito: (a).

1. Ponto-chave. A identidade \(e(p, v(p, m)) = m\) é tautológica: se \(v\) é o máximo de \(u\) sob \(p\cdot x \leq m\), e \(e\) é o mínimo de \(p\cdot x\) para atingir \(v\), o gasto é exatamente \(m\) — o mesmo ponto geométrico.

2. Derivação. Marshalliana CD simétrica (\(\alpha = 1/2\)): \(x_1^M = 0{,}5\cdot 16/2 = 4\); \(x_2^M = 0{,}5\cdot 16/8 = 1\). Walras: \(2(4) + 8(1) = 16\). ✓ Utilidade: \(v = \sqrt{4\cdot 1} = 2\). EMP com \(\bar u = 2\): tangência \(x_2/x_1 = p_1/p_2 = 1/4\); com \(\sqrt{x_1 x_2} = 2\): \(x_2 = x_1/4\), \(\sqrt{x_1\cdot x_1/4} = x_1/2 = 2\) ⇒ \(x_1 = 4, x_2 = 1\). Gasto: \(2(4) + 8(1) = 16 = m\). ✓

3. Armadilha. (b) e (d) tentam induzir a pensar em fator multiplicativo extra na identidade. (c) confunde \(v\) com \(m\) — erro canônico de quem não distingue "utilidade" de "moeda".

4. Extensão. Essa identidade é o contrato operacional da dualidade. Cada passagem entre UMP e EMP faz a calibração automática; os dois problemas são o mesmo ponto por construção.

5. Peer-review. A questão exige executar UMP + EMP num caso limpo. Quem sai sem confusão, tem a dualidade no corpo; se errar, é diagnóstico de que "identidade recíproca" ainda soa como fórmula, não como tautologia.

Q4. 🟡 Dada a função indireta \(v(p_1, p_2, m) = \ln m - \tfrac{2}{3}\ln p_1 - \tfrac{1}{3}\ln p_2\) (forma log-linear), aplicando Roy, a demanda Marshalliana \(x_1^M\) é:

Gabarito: (d).

1. Ponto-chave. Roy: \(x_i^M = -(\partial v/\partial p_i)/(\partial v/\partial m)\). Funciona em qualquer transformação monótona de \(v\) — inclusive em log.

2. Derivação. \(\partial v/\partial p_1 = -(2/3)/p_1\); \(\partial v/\partial m = 1/m\). Logo \(x_1^M = -(-(2/3)/p_1)/(1/m) = (2/3)\cdot m/p_1\).

3. Armadilha. (a) troca os expoentes de \(p_1\) e \(p_2\) — erro de reconhecimento. (b) é CD com \(\alpha = 1\). (c) esquece de dividir por \(\partial v/\partial m\) — pega só o numerador.

4. Extensão. A forma log-linear de \(v\) é a assinatura de preferências CD. Verificamos: \(v = \ln m - (2/3)\ln p_1 - (1/3)\ln p_2 = \ln(m/(p_1^{2/3} p_2^{1/3}))\) — transformação monotônica de \(m\cdot \phi(p) = m\cdot p_1^{-2/3} p_2^{-1/3}\). Representa preferência CD com expoentes \((2/3, 1/3)\).

5. Peer-review. Roy em log é a fórmula que vai para o código em AIDS/PIGLOG. Aluno que não aplica em 30 segundos ainda não está fluente na forma estrutural.

Q5. 🟡 Qual das afirmações abaixo é verdadeira sobre propriedades estruturais da dualidade?

Gabarito: (b).

1. Ponto-chave. Por Shephard, \(h_i = \partial e/\partial p_i\). A derivada de uma função homogênea de grau 1 em \(p\) é homogênea de grau 0 em \(p\). Daí o grau 1 de \(e\) e o grau 0 de \(h\).

2. Derivação. \(e(\lambda p, \bar u) = \lambda e(p, \bar u)\) (grau 1) — escala de preços escala o gasto. Diferenciando em \(p_i\): \(\lambda \cdot h_i(\lambda p, \bar u) = \lambda\cdot h_i(p, \bar u)\), logo \(h_i(\lambda p, \bar u) = h_i(p, \bar u)\) — grau 0 como afirmado.

3. Armadilha. (a) inverte graus. (c) é falso — \(v\) é homogênea grau 0 em \((p, m)\) conjuntamente, não grau 1 em \(p\) só. (d) troca sinal: a matriz é semidef. negativa (concavidade de \(e\)).

4. Extensão. Homogeneidade grau 0 de \(h\) é operacionalmente importante para Slutsky: garante que \(Sp = 0\), uma das três restrições estruturais da matriz de Slutsky.

5. Peer-review. Questão de reconhecimento. Se o aluno hesitou entre (b) e (d), ainda não consolidou sinal de concavidade. A tarefa pedagógica: fixar que concavidade ⇒ Hessiano semidef. negativo; convexidade ⇒ positivo. Sem isso, Slutsky na Aula 3 vira salada.

Q6. 🔴 A partir da identidade \(x_i^M(p, m) = h_i(p, v(p, m))\), aplicando \(\partial/\partial p_j\) em ambos os lados e usando Roy, deriva-se a Equação de Slutsky. Qual é a forma correta?

Gabarito: (b).

1. Ponto-chave. Slutsky sai da dualidade em 3 linhas: derivar identidade em \(p_j\), usar Roy para substituir \(\partial v/\partial p_j\), e simplificar usando \(\partial h_i/\partial \bar u = (1/\lambda^*)\cdot \partial x_i^M/\partial m\).

2. Derivação. Começando de \(x_i^M(p, m) = h_i(p, v(p, m))\), derivar em \(p_j\): \[ \frac{\partial x_i^M}{\partial p_j} = \frac{\partial h_i}{\partial p_j}\bigg|_{\bar u = v} + \frac{\partial h_i}{\partial \bar u}\cdot \frac{\partial v}{\partial p_j}. \] Por Roy, \(\partial v/\partial p_j = -\lambda^* x_j^M\). Pela segunda identidade dual, \(h_i(p, \bar u) = x_i^M(p, e(p, \bar u))\), derivar em \(\bar u\): \(\partial h_i/\partial \bar u = (\partial x_i^M/\partial m)\cdot (\partial e/\partial \bar u) = (\partial x_i^M/\partial m)/\lambda^*\) (pois \(\partial e/\partial \bar u = 1/\lambda^*\)). Substituir: \[ \frac{\partial x_i^M}{\partial p_j} = \frac{\partial h_i}{\partial p_j} + \frac{1}{\lambda^*}\cdot \frac{\partial x_i^M}{\partial m}\cdot (-\lambda^* x_j^M) = \frac{\partial h_i}{\partial p_j} - x_j^M\cdot \frac{\partial x_i^M}{\partial m}.\ \blacksquare \]

3. Armadilha. (a) inverte índices: \(x_i^M\) em vez de \(x_j^M\) no efeito-renda. Slutsky é assimétrica: efeito de \(p_j\) sobre \(x_i\) tem renda proporcional ao consumo do bem cujo preço mudou, isto é, \(x_j\). (c) erra o sinal do efeito-renda. (d) erra sinal do efeito-substituição.

4. Extensão. Em forma de elasticidades: \(\varepsilon_{ij} = \varepsilon^c_{ij} - s_j\eta_i\). Essa é a identidade que ANPEC e qualifiers de mestrado exigem na cabeça. Decompor Marshalliana em Hicksiana (substituição compensada) e efeito-renda é o coração de Slutsky.

5. Peer-review. Questão no fronteira da Aula 2/Aula 3. Incluí aqui para o aluno sair capaz de derivar Slutsky no caderno dele. Se precisar ficar colando na fórmula decorada, não aprendeu dualidade de verdade; se derivou sozinho, dominou.

Q7. 🔴 A matriz \(S\) com entradas \(S_{ij} = \partial h_i(p, \bar u)/\partial p_j\) herda três propriedades estruturais da função gasto \(e(p, \bar u)\). Quais?

Gabarito: (c).

1. Ponto-chave. Três propriedades, três teoremas de análise: Young, concavidade, Euler.

2. Derivação. (i) Por Shephard, \(S_{ij} = \partial^2 e/\partial p_j\partial p_i\) — cruzadas iguais por Young ⇒ simetria. (ii) Concavidade de \(e\) em \(p\) ⇒ Hessiano semidef. negativo. (iii) \(e\) é homogênea grau 1 em \(p\), logo \(h\) (=\(\partial e/\partial p\)) é grau 0. Euler em \(e\): \(\sum_j p_j\, h_j = e\). Derivando em \(p_i\): \(h_i + \sum_j p_j S_{ji} = h_i\) ⇒ \(\sum_j p_j S_{ji} = 0\), isto é \(Sp = \mathbf{0}\).

3. Armadilha. (a) inverte sinal. (b) inventa \(Sp = \mathbf{1}\). (d) esquece simetria — a matriz compensada é simétrica; a não-compensada (Marshalliana, \(\partial x^M/\partial p\)) é que pode ser assimétrica.

4. Extensão. Essas 3 propriedades são os três testes empíricos da teoria da demanda. Violações (simetria, semidef. neg., adição) indicam: má especificação, viés de agregação, ou preferências incoerentes (não representáveis por \(u\)). GARP (Varian 1982) é o equivalente discreto desse teste.

5. Peer-review. Questão exigente — pede três coisas com três justificativas distintas. O aluno que memoriza só "simetria e semidef. neg." e esquece \(Sp = 0\) não captou a estrutura. O \(Sp = 0\) é o elemento menos lembrado em N&S (raramente enfatizado); J-R §2.1 discute.

Q8. 🔴 Walras (\(p\cdot x^M = m\)) combinada com homogeneidade grau 0 de \(x^M\) gera duas restrições de adição em elasticidades. Quais?

Gabarito: (d).

1. Ponto-chave. Engel (soma ponderada de elasticidades-renda = 1) nasce de derivar Walras em \(m\). Cournot (soma ponderada de elasticidades-preço cruzadas dá menos a parcela) nasce de derivar Walras em \(p_j\).

2. Derivação.

Engel (derivada em \(m\)). Parte de Walras: \(\sum_i p_i x_i^M(p, m) = m\). Derive em \(m\): \[ \sum_i p_i\,\frac{\partial x_i^M}{\partial m} = 1. \] Multiplique cada termo por \(\frac{x_i^M}{x_i^M}\cdot \frac{m}{m}\), reagrupando como \(s_i \cdot \eta_i\): \[ \sum_i \underbrace{\frac{p_i x_i^M}{m}}_{s_i}\cdot \underbrace{\frac{m}{x_i^M}\cdot \frac{\partial x_i^M}{\partial m}}_{\eta_i} = 1 \;\Longrightarrow\; \sum_i s_i\eta_i = 1. \]

Cournot (derivada em \(p_j\)). Derive Walras em \(p_j\): \[ x_j^M + \sum_i p_i\,\frac{\partial x_i^M}{\partial p_j} = 0. \] Divida tudo por \(m\): \[ \frac{x_j^M}{m} \;+\; \sum_i \frac{p_i}{m}\,\frac{\partial x_i^M}{\partial p_j} = 0. \] No primeiro termo, \(x_j^M/m = s_j/p_j\) (basta multiplicar por \(p_j/p_j\)) — mas preferimos \(s_j\), então multiplique por \(p_j\) os dois lados da última identidade e nomeie \(s_j = p_j x_j^M/m\); simultaneamente, no segundo termo, multiplique e divida cada parcela por \(p_j\cdot x_i^M\) para formar \(s_i\) e \(\varepsilon_{ij}\): \[ s_j \;+\; \sum_i \underbrace{\frac{p_i x_i^M}{m}}_{s_i}\cdot \underbrace{\frac{p_j}{x_i^M}\,\frac{\partial x_i^M}{\partial p_j}}_{\varepsilon_{ij}} = 0 \;\Longrightarrow\; \sum_i s_i\varepsilon_{ij} = -s_j. \]

3. Armadilha. (a) erra unidade em Engel ("\(= m\)" mistura elasticidade com moeda). (b) esquece peso \(s_i\) — mistura "derivada" com "elasticidade". (c) simplifica Cournot para \(-1\) ignorando \(s_j\).

4. Extensão. Em AIDS e PIGLOG, Engel e Cournot são impostas na parametrização. Violações amostrais disparam diagnóstico: (i) má especificação funcional; (ii) viés de agregação; (iii) preferências não-maximizadoras (comportamental). Questão de prova ANPEC quase todo ano.

5. Peer-review. Restrições de adição são consequência trivial de Walras — por isso "qualquer" modelo de demanda razoável as satisfaz. Elas servem como testes de sanidade: dados que violam são ou ruidosos ou refutam o modelo.

Q9. 🔴 Seja \(u\) contínua, monotônica, quasicôncava e homotética. A função gasto fatoriza como \(e(p, \bar u) = c(p)\cdot f(\bar u)\) com \(f\) crescente e \(c(\cdot)\) específica. Qual é a propriedade estrutural de \(c(p)\)?

Gabarito: (c).

1. Ponto-chave. Homotetia ⇒ \(h(p, \bar u) = f(\bar u)\cdot h(p, 1)\) (demanda radial em \(\bar u\)). Logo \(e(p, \bar u) = p\cdot h = f(\bar u)\cdot p\cdot h(p, 1) = f(\bar u)\cdot c(p)\), com \(c(p) = e(p, 1)\). \(c\) herda homogeneidade grau 1 e concavidade em \(p\) de \(e\).

2. Derivação. Como \(c(p) = e(p, 1)\), \(c(\lambda p) = e(\lambda p, 1) = \lambda\,e(p, 1) = \lambda c(p)\) (grau 1). Concavidade: \(e\) é côncava em \(p\) para cada \(\bar u\), incluindo \(\bar u = 1\). Logo \(c\) é côncava.

3. Armadilha. (a) inverte sinal. (b) esquece que \(e\) é sempre côncava em \(p\) — logo \(c\) é. (d) confunde homogeneidade com propriedade de \(f\).

4. Extensão. \(c(p)\) é a fundação do índice de preço de Konüs. A razão \(c(p^1)/c(p^0)\) é o custo-de-vida estrutural exato (mesmo nível de utilidade, dois vetores de preços). IPCA/CPI aproximam isso via Laspeyres/Paasche/Fisher. Essa aproximação tem viés bem-conhecido que Konüs quantifica em preferências não-homotéticas.

5. Peer-review. Questão conceitual importante. Aparece em J-R §2.1 com demonstração; em N&S é passada mais ligeiramente. Útil para quem quer fazer índice de preços ou economia de bem-estar (Harberger triangles).

Q10. 🔴🔴 (difícil mesmo para 🔴 — exige raciocinar na recíproca da dualidade)
Um economista empírico observa \(e(p, \bar u)\) (por exemplo, consegue medir o gasto mínimo para atingir um padrão de vida \(\bar u\) via cesta de referência). É possível recuperar a preferência \(\succeq\) (e, equivalentemente, \(u\) a menos de transformação monótona) a partir de \(e\) sozinha?

Gabarito: (b).

1. Ponto-chave. Dualidade é um mapa bijetivo entre preferências bem-comportadas e funções \(e\) que satisfazem os axiomas estruturais (E1–E5). Dada \(e\), recupera-se a preferência uniquamente. É o teorema da recíproca (J-R §2.1).

2. Derivação (esboço). O conjunto superior de nível \(U(\bar u) = \{x : u(x)\geq \bar u\}\) é fechado e convexo (sob quasiconcavidade). Pelo teorema de separação (Hahn-Banach / Minkowski), cada conjunto convexo fechado é a intersecção dos semi-espaços suportes. O semi-espaço suporte em \(p\) é \(H(p, \bar u) = \{x : p\cdot x \geq e(p, \bar u)\}\), pois \(e\) é o menor valor \(p\cdot x\) sobre \(U(\bar u)\). Logo: \[ U(\bar u) = \bigcap_{p\gg 0}\{x : p\cdot x \geq e(p, \bar u)\}. \] Conhecido \(U(\bar u)\) para cada \(\bar u\), a preferência está caracterizada (relação de nível é tudo o que importa ordinalmente).

3. Armadilha. (a) é parcialmente correto — Shephard recupera \(h\) sim, mas "integrar \(h\)" para obter \(u\) é técnico e exige simetria de Slutsky compensada (automática se \(e\) satisfaz as propriedades). Não é o argumento canônico do teorema. (c) é o erro de quem pensa em \(e\) como "resumo" em vez de "caracterização completa". (d) restringe desnecessariamente: recíproca vale para qualquer preferência convexa quasicôncava, homotética ou não.

4. Extensão. Esse teorema é a justificativa teórica de estimação estrutural via função gasto. AIDS parametriza \(\ln e(p, \bar u)\) e extrai demanda via Shephard (consistente com teoria). A recuperação de \(u\) a partir de \(\hat{e}\) estimada é o que torna AIDS compatível com welfare analysis rigoroso.

5. Peer-review. 🔴🔴 porque testa internalização de recíproca da dualidade — não é derivada em aula presencial (só mencionada). J-R §2.1 dá o argumento em 1 página; N&S omite. Esse conteúdo separa quem entendeu dualidade como simetria de quem memorizou "UMP dá \(x^M\), EMP dá \(h\)". A armadilha mais sutil é (c) pelo ceticismo performático — rejeitar por incapacidade de recuperar \(u\) exatamente, quando o teorema já entrega a preferência (ordinal, que é o que importa).
