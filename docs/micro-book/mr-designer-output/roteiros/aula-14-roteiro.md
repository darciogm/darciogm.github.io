# Aula 14 — Lab: Isoquantas, TMST e Rendimentos de Escala
**Tipo**: P (Prática Ativa)
**Capítulo(s)**: Cap. 9 (Seções 9.5–9.7)
**Learning Outcomes desta aula**:
1. **Manipular** parâmetros de funções de produção e observar efeitos sobre isoquantas
2. **Ensinar** propriedades de uma função de produção para colegas (Jigsaw)
3. **Diagnosticar** o tipo de tecnologia a partir de dados observados

## Em Aula (100 min)

### Abertura (10 min)
- **Check-in**: Quiz + dúvidas
- **Organização Jigsaw**: mesma dinâmica do Aula 04, agora para produção
  - Grupo A: Linear (q = aL + bK)
  - Grupo B: Leontief (q = min{aL, bK})
  - Grupo C: Cobb-Douglas (q = AL^α K^β)
  - Grupo D: CES (q = A(αL^ρ + βK^ρ)^(1/ρ))

### Bloco 1: Jigsaw — Fase de Especialistas (25 min)
**Estratégia ativa**: Jigsaw

**Cada grupo de especialistas produz "Ficha Técnica da Tecnologia"** (meia A3):
1. Forma funcional + parâmetros
2. Mapa de isoquantas (desenho)
3. TMST (fórmula + comportamento)
4. Elasticidade de substituição σ
5. Rendimentos de escala (condição)
6. Para que setores/indústrias se aplica (exemplo real brasileiro)
7. PMg_L e PMg_K (fórmulas)
8. "Quando esta tecnologia NÃO é realista?"

**Materiais**: Folhas A3, canetões, acesso ao livro digital

### Bloco 2: Jigsaw — Fase de Ensino Mútuo (25 min)
**Fase 2 — Grupos mistos** (1 de cada função):
- Cada especialista: 5 min para ensinar usando ficha técnica
- Ao final: grupo constrói tabela comparativa das 4 tecnologias
- Questão de síntese: "Ordene as 4 funções do menos ao mais flexível. Justifique."

### Bloco 3: Lab Interativo — "Diagnóstico Tecnológico" (25 min)
**Estratégia ativa**: Simulação com gráficos interativos + gamificação

**Rodada 1-4** (5 min cada):
Professor projeta dados de produção (L, K, q) de 4 "firmas misteriosas":

**Firma 1**: L = {1,2,4}, K = {1,2,4}, q = {1,2,4}
→ "Que tipo de tecnologia? CRS, linear? CD com α+β=1?"

**Firma 2**: L = {1,2,3}, K = 10 (fixo), q = {10, 14, 17}
→ "PMg decrescente no CP. Que função gera isso?"

**Firma 3**: Se dobrar ambos os insumos, q triplica
→ "IRS! α + β > 1 na CD"

**Firma 4**: Isoquantas são linhas retas com inclinação −2
→ "Linear: q = 2L + K"

**Dinâmica**: Grupos mistos discutem (3 min) → votam → professor revela

**Rodada Bônus**: Professor projeta dados reais de agricultura brasileira (IBGE):
- Hectares de terra, horas de trabalho, toneladas de soja
- "Que tipo de rendimentos de escala tem a produção de soja no Brasil?"

### Fechamento (15 min)
- **Debrief**: Tabela comparativa completa projetada — turma valida
- **Minute paper**: "Qual tecnologia melhor descreve o setor onde você quer trabalhar?"
- **Preview**: "Agora que sabemos COMO a firma produz, vamos ver QUANTO CUSTA produzir — custos de produção"
