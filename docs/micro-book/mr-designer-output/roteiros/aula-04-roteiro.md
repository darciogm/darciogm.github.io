# Aula 04 — Lab: Curvas de Indiferença e TMS
**Tipo**: P (Prática Ativa)
**Capítulo(s)**: Cap. 3 (Seções 3.4–3.6)
**Learning Outcomes desta aula**:
1. **Comparar** propriedades de diferentes funções utilidade (linear, Leontief, Cobb-Douglas, CES)
2. **Ensinar** para colegas as características de uma função utilidade específica (Jigsaw)
3. **Identificar** a função utilidade a partir de seu mapa de indiferença

---

## Pré-Aula (Responsabilidade do Aluno)
**Tempo estimado**: 40 min
**Plataforma**: LMS + Livro digital interativo

### Leitura obrigatória
- Cap. 3, Seções 3.4–3.6 (quase-concavidade, preferências especiais, elasticidade de substituição)
- Focar na função que escolheu para o Jigsaw

### Atividade pré-aula
- [ ] Estudar a fundo a função utilidade que escolheu: derivar TMS, desenhar curvas, identificar elasticidade de substituição
- [ ] Preparar 3 pontos-chave para ensinar aos colegas
- [ ] Responder Quiz Pré-Aula no LMS (prazo: 2h antes da aula)

### Quiz Pré-Aula (5 questões)
1. Para U = min{2x, 3y}, qual é a proporção ótima de consumo x/y? (Bloom 3)
2. O que a elasticidade de substituição σ mede? (Bloom 2)
3. Quando σ → ∞, as curvas de indiferença se aproximam de que formato? (Bloom 2)
4. A função U = x^0.3 · y^0.7 é quase-côncava? O que isso implica sobre as curvas de indiferença? (Bloom 2)
5. Dê um exemplo de bens que seriam modelados como complementares perfeitos. (Bloom 1)

---

## Em Aula (100 min)

### Abertura (10 min)
- **Check-in**: Resultados do quiz + dúvidas rápidas
- **Organização dos grupos Jigsaw**: 4 funções × grupos de 4-5 alunos
  - Grupo A: Linear (substitutos perfeitos)
  - Grupo B: Leontief (complementares perfeitos)
  - Grupo C: Cobb-Douglas
  - Grupo D: CES

### Bloco 1: Jigsaw — Fase de Especialistas (25 min)
**Estratégia ativa**: Jigsaw (Aronson)
**Fase 1 — Grupos de especialistas**:
- Alunos do mesmo tema se reúnem (todos os "A" juntos, todos os "B" juntos, etc.)
- Cada grupo de especialistas deve produzir um **"cartão de referência"** (meia folha A4) com:
  1. Forma funcional + parâmetros
  2. Mapa de indiferença (desenho)
  3. TMS (fórmula + interpretação)
  4. Elasticidade de substituição σ
  5. Para que tipos de bens se aplica (exemplo real)
  6. "Pegadinha" — erro comum ao trabalhar com essa função

**Materiais**: Folhas A4, canetões coloridos, acesso ao livro digital para consulta dos gráficos interativos

### Bloco 2: Jigsaw — Fase de Ensino Mútuo (25 min)
**Fase 2 — Grupos mistos**:
- Reagrupar: cada novo grupo tem 1 especialista de cada função (1 de A, 1 de B, 1 de C, 1 de D)
- Cada especialista tem **5 minutos** para ensinar sua função ao grupo usando o cartão de referência
- Regra: quem está ensinando não pode ser interrompido nos primeiros 3 min; perguntas nos últimos 2 min
- Ao final: grupo misto discute — "Qual função é mais flexível? Qual é mais restritiva? Por quê?"

### Bloco 3: Mini-Torneio — "Identifique a Função" (25 min)
**Estratégia ativa**: Gamificação + Quiz visual
**Dinâmica**:
- Professor projeta sequencialmente 8 mapas de indiferença (gráficos do livro)
- Para cada mapa, grupos mistos têm 90 segundos para:
  1. Identificar a função utilidade
  2. Estimar parâmetros-chave
  3. Escrever a TMS naquele ponto
- Cada rodada: resposta em plaquinha/app → pontuação imediata

**Mapa 1**: Linhas retas com inclinação −2 → Linear U = 2x + y
**Mapa 2**: L's com vértice em y = 2x → Leontief U = min{x, y/2}
**Mapa 3**: Hipérboles "padrão" → Cobb-Douglas U = x^0.5 · y^0.5
**Mapa 4**: Curvas entre Leontief e linear → CES com σ intermediário
**Mapa 5-8**: Variações com parâmetros diferentes + 1 "pegadinha" (preferências não-monotônicas)

**Ranking**: Projetar classificação após cada rodada

### Fechamento (15 min)
- **Debrief**: Projetar tabela comparativa completa das 4 funções — turma preenche coletivamente
- **Minute paper**: "Qual função foi mais difícil de entender? Por quê?"
- **Preview Aula 05**: "Agora que sabemos representar preferências, vamos ver como o consumidor ESCOLHE — restrição orçamentária + ótimo. Cap. 4."

## Pós-Aula (Responsabilidade do Aluno)
**Tempo estimado**: 50 min
**Plataforma**: LMS
**Prazo**: Até 48h antes da Aula 05

### Atividade pós-aula
- [ ] **Problem set 1** — Preferências e utilidade (6 exercícios):
  1. Verificar axiomas para 2 relações de preferência
  2. Provar que V = ln(U) representa as mesmas preferências que U = x^a · y^(1-a)
  3. Calcular TMS para CES com ρ = 0.5 e ρ = −1
  4. Dado mapa de indiferença, identificar função e estimar parâmetros
  5. Provar que Cobb-Douglas é caso limite da CES quando ρ → 0
  6. Desenhar curvas para preferências não-convexas e explicar implicações
- [ ] **Reflexão** (2-3 frases): "Quais bens do seu dia-a-dia são mais bem modelados como substitutos? E como complementares?"
- [ ] **Leitura preparatória para Aula 05**: Cap. 4, Seções 4.1–4.3 (restrição orçamentária, escolha ótima, condição de tangência)
