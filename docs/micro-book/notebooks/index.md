# Cadernos Computacionais

Estes cadernos Python acompanham o livro e permitem explorar os modelos microeconômicos interativamente. Cada caderno pode ser executado no [Google Colab](https://colab.research.google.com/) ou localmente com Python 3.

**Requisitos:** `numpy`, `matplotlib`, `scipy`

```bash
pip install numpy matplotlib scipy
```

| Caderno | Capítulos | Temas |
|---------|-----------|-------|
| [Teoria do Consumidor](consumidor.py) | 3-6 | Utilidade, otimização, Slutsky, elasticidade |
| [Incerteza e Comportamento](incerteza.py) | 7-8 | Utilidade esperada, prospect theory, desconto |
| [Teoria dos Jogos](jogos.py) | 9a-9d | Nash, Cournot, leilões, jogos repetidos |
| [Firma e Custos](firma.py) | 10-12 | Produção, custos, oferta competitiva |
| [Mercados](mercados.py) | 13-16 | Tributação, equilíbrio geral, monopólio, oligopólio |
| [Fatores e Tempo](fatores.py) | 17-18 | Oferta de trabalho, VPL, Hotelling |
