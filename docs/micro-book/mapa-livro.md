# Mapa do Livro

O diagrama abaixo mostra as dependências entre capítulos. Setas indicam pré-requisitos: siga as setas para planejar sua leitura. Passe o mouse sobre um capítulo para ver suas conexões.

<iframe src="graficos/mapa-livro.html" title="Mapa de dependências entre capítulos" style="width:100%;height:700px;border:1px solid #ddd;border-radius:6px;"></iframe>

**Como ler o mapa:**

- Capítulos à esquerda são mais fundamentais; à direita, mais avançados
- Cores indicam a parte do livro (ver legenda no diagrama)
- Siga qualquer caminho de setas para uma sequência de leitura válida
